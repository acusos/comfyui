# ComfyUI Configuration

This repository contains the configuration, workflows, documentation and support
scripts for my ComfyUI installation.

## Environment

| Item | Value |
|------|-------|
| Host | ai (192.168.20.116) |
| OS | Ubuntu 24.04 LTS |
| GPU | NVIDIA GeForce RTX 5090 32GB |
| ComfyUI | /var/lib/docker/ComfyUI |
| Models | /models/comfyui |

## Repository Layout

config/        Configuration files

workflows/     Exported workflows

scripts/       Utility scripts

docs/          Documentation

benchmarks/    Performance results

This repository does **not** contain:

- ComfyUI source code
- Model files
- Python virtual environments
- Generated images or videos

Those are maintained separately.
